# EmployeeManagementSystem
Employee Management System ASP.NET MVC .Net6 Project  

#Technologies
1) Front-end
    - HTML5
    - CSS
    - JS
2) Back-end
    - ASP.NET MVC
3) Database
    - Microsofe SQL Server

#Features
1) Display the Tasks For Employee
2) Admin Send Reports To Employee
3) Employees Receive the Reports in Mailbox
4) Employee Can Take Attendance By StartWork & EndTask Request Buttons
5) Employee Can Take Vacations
6) Admin Displays the Employee Contacts , Tasks , Vacation Requests , Attendance
7) Admin Only Can Register Accounts For Employee Or Employees
8) All User Can Login & Start with his Own Dashboard
